## Getting Started

### Installation

1. Clone the repository

   ```
   git clone https://github.com/Passant-Abdelgalil/coach-finder.git
   ```

2. Navigate to the directory

   ```
   cd coach-finder
   ```

3. Install dependencies

   ```
   npm i
   ```

### Compiles and hot-reloads for development

```
npm run serve
```
